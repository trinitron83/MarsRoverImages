# API to get images from Mars Rover

## Methods
### /api/getimagesforday

#### Parameters: 

* dateParam - date of images

* page - paging number of images to return. defaults to 0

#### Returns: 

* image data in byte[] format

### /api/getimageurls
#### Parameters: 

* dateParam - date of images

#### Returns: 

* URL addresses for each image for the day
